# Modern JavaScript: ES6 and Beyond — Learning Laboratory

A structured, self-directed study repository documenting my progression through modern JavaScript concepts. Each topic is treated as an independent lab, covering theoretical foundations, annotated code examples, and hands-on practice problems.

---

## Overview

This repository serves as a personal knowledge base built alongside the
[Modern JavaScript ES6+ Tutorial](https://youtu.be/yyrq_8-bk0s). It is organized to reflect how I approach learning: understanding the theory first, working through practical examples, and reinforcing concepts through deliberate problem-solving.

The goal is not just to follow a tutorial, but to build a reference-grade resource that demonstrates depth of understanding in modern JavaScript development.

---

## Repository Structure

Every topic follows a consistent three-part structure:

```
topic-name/
├── README.md            # Theory, concepts, and personal notes
├── examples.js          # Annotated code examples from the tutorial
└── practice/
    ├── problems.md      # Practice problems
    └── solutions.js     # My implemented solutions
```

This separation of theory, examples, and practice is intentional. It mirrors how concepts are applied in professional development — understanding the why before writing the code.

## Topics Covered

**Core Language Syntax**
Variables with `let` and `const`, template literals, arrow functions, rest and spread operators, enhanced object literals, and destructuring for both arrays and objects.

**Object-Oriented Programming**
Class-based OOP in JavaScript including constructors, methods, encapsulation, and class inheritance using `extends` and `super`.

**Modules**
ES6 module system using `import` and `export`, named and default exports, and organizing code across files.

**Asynchronous JavaScript**
The full asynchronous progression — from callbacks to Promises, `Promise.all` for concurrent operations, AJAX for server communication, the Fetch API for HTTP requests, and `async/await` for clean asynchronous code.

**Advanced Concepts**
Symbols as unique identifiers, custom iterators using the iterator protocol, generator functions with `yield`, strict mode for safer code, and structured error handling with `try/catch/finally`.

## Local Setup

```bash

# Navigate into the project

# Open in VS Code
code .
```

